github里如何删除一个repository仓库

     高手请绕行，新手往下走。

     作为一个刚接触github（https://github.com/）的新手，除了感叹开源的丰富和强大之外，自己肯定也想试用一下，因此申请帐号神马的。今天自己创建一个Repository测试，创建过程好简单的说。但是想删除刚创建的Repository，一时竟然找不到，甚至都动用了Google，结果查得到的都是之前在较老github上的删除方式，比如什么Admin按钮的根本就没有了，好在自己还是摸索到了（人笨，这还需要摸索！！！），原来还是很简单，整理如下：

一、点击进入你要删除的repository，如下图，点击右侧的Settings；

点击Settings之后，进入到该Repo的设置页面。

二、在该Repo的Settings页面最下方有Delete this repository 按钮，点击；

三、在弹出的对话框中输入你的repository的名字，确认删除；

注意上面的一句话：This action CANNOT be undone。也就是删除操作说不可撤销、不可恢复。

四、最后一步，点击下方的按钮完成删除；

好吧，就这几步，就完全删除了一个项目。
这么简单的事还记下来，是因为肯定还有我这样的新手不知道从哪删除，还去Baidu、Google而不得。
